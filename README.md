# IU_TV4

### About the program
The code implements a container with isolated pid, mnt and net. Programming language used is C. Disk image is presented under disk.img name. The mnt flie contains alpine, mnt and oldroot directories.

### Quick start
Prerequirenments install docker and pull aplene image
```
gcc main.c -o main
sudo ./main
```
